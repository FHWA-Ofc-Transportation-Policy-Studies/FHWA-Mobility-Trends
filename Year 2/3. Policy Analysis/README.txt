Code and data within this folder allows the user to create forecasts for the performance metric through 2050 using the champion models built in the modeling section of this repo. 

To create a forecast, first, run the forecast code associated with the performance metric you are interested in. The final line of code will output such as 'VMT Forecast Dataset and Results' file.

Second, use the 'forecast output' of the code and put them into the chosen performance metric's excel spreadsheet in the associated performance metric file in the Forecasting_Analysis folder.
Following the directions in this excel spreadsheet will allow you to create the forecast with a pivot table to see the results.
